Occurrences API

Available Endpoints

This services provides an API to look up number of records (occurrences) indexed with a concept in a database or a list of concepts to are also used to index these records (co-occurrences). See also sources and issue tracker.

Concept Schemes

  • GET /voc - Returns a list of supported vocabularies (concept schemes) (documentation)

Databases

  • GET /databases - Returns a list of supported databases (documentation)

Occurrences

  • GET /occurrences - Returns occurrences or co-occurrences (documentation)
